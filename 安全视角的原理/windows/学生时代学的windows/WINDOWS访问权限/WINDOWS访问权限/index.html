WINDOWS访问权限

安全对象与安全描述符

ACL

ACL，Access Control List，即访问控制列表，用来表示用户（组）权限的列表。 它分为DACL和SACL

DACL:自由访问控制列表

DACL是安全对象的一个属性，当一个进程试图访问一个安全对象时，系统会检查DACL，进而判断进程是否能正常访问此对象。

SACL:系统访问控制列表
